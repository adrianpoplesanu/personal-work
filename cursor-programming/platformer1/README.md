# 2D Platformer Game

A 2D platformer game built with Python and Pygame featuring a main character and a companion that follows the player around.

## Features

- **Player Character**: Blue character that can move left/right and jump on platforms
- **Companion**: Yellow character that automatically follows the player around
- **Physics**: Gravity, collision detection, and platform jumping
- **Multiple Platforms**: Various platforms to jump between

## Setup

1. **Activate the virtual environment**:
   ```bash
   source venv/bin/activate  # On macOS/Linux
   # or
   venv\Scripts\activate  # On Windows
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the game**:
   ```bash
   python game.py
   ```

## Controls

- **Left Arrow / A**: Move left
- **Right Arrow / D**: Move right
- **Space / Up Arrow**: Jump

## Game Structure

- `game.py`: Main game file containing all game logic
- `requirements.txt`: Python dependencies
- `venv/`: Virtual environment directory

Enjoy playing!
